# Healthcare Appointment System

A production-ready React + TypeScript frontend for managing healthcare appointments.

## Stack

- React + TypeScript
- Vite
- TailwindCSS
- React Router
- Axios
- Zustand

## Features

- Role-based login and signup for patients and doctors
- Doctor listing, filtering, and profile pages
- Appointment booking with collision prevention
- Appointment management with cancel and reschedule flows
- Patient and doctor dashboards
- Mock Axios API layer with local persistence

## Scripts

```bash
npm install
npm run dev
```

```bash
npm run build
```
